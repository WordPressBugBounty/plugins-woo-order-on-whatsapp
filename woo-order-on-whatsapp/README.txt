Add a WhatsApp order button to your WooCommerce store. Completely free — and if you want automatic status notifications to customers, MyD Notifications has you covered.

== Description ==

**Order on Mobile for WooCommerce** connects your WooCommerce store directly to WhatsApp — the messaging app your customers already use every day.

Add a WhatsApp button to your product and cart pages, redirect customers to WhatsApp after checkout, and keep them in the loop with order updates. Everything your store needs to sell and communicate via WhatsApp, completely free.

### Why Order on Mobile for WooCommerce?

- **Higher conversions** – a single tap sends the order. Less friction, more sales.
- **Direct communication** – answer questions, upsell, and confirm orders in real time.
- **Completely free** – no locked features, no paid plan required to use the plugin.
- **Optional automatic notifications** – want WhatsApp status updates sent to customers automatically? [MyD Notifications](https://myddelivery.com/notifications/) adds that on top.

### Free Features

- WhatsApp button on product pages
- WhatsApp button on cart page
- Redirect customer to WhatsApp after checkout with full order summary
- Dynamic message templates with smart tags ({order_id}, {total}, {customer_name}, and more)
- Variable product support — accurate pricing and details for every variation
- Customizable message content for every touchpoint
- Hide price / Add to Cart — perfect for catalog-mode or "call for price" stores
- Elementor widget — drag and drop the button anywhere on your site
- Device visibility control — show the button only on mobile if you prefer
- Works with any WordPress theme

### Automatic WhatsApp Notifications with MyD Notifications

The free plugin already handles WhatsApp communication at checkout. But if you want your customers to receive automatic status updates — confirmed, preparing, out for delivery, delivered — that's where **[MyD Notifications](https://myddelivery.com/notifications/)** comes in.

MyD Notifications is a paid add-on that works natively with this plugin. Every time an order status changes in WooCommerce, the matching WhatsApp message fires automatically. No manual work, no extra setup beyond an API key from the MyD dashboard.

[See MyD Notifications plans and pricing →](https://myddelivery.com/notifications/)

== Installation ==

1. Install directly from the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the Order on Mobile for WooCommerce settings page.
4. Enter your WhatsApp number and configure your message templates.
5. (Optional) Connect [MyD Notifications](https://myddelivery.com/notifications/) to send automatic order status updates to your customers.

== Frequently Asked Questions ==

= Is this plugin free? =
Yes. All core features are completely free — WhatsApp buttons, checkout redirect, dynamic message templates, Elementor widget, and more. No paid plan required.

= What requires MyD Notifications? =
Two features require a MyD Notifications subscription: automatic alerts to your store WhatsApp when a new order is placed, and automatic WhatsApp notifications to customers when order status changes. Everything else is free. [Learn more →](https://myddelivery.com/notifications/)

= Do my customers need to install anything? =
No. Messages go to the WhatsApp they already have on their phone. No app install, no account, no friction.

= Does it work with variable products? =
Yes. The WhatsApp message includes the correct variation details and pricing automatically.

= What are the dynamic message tags? =
You can use smart tags like {order_id}, {total}, {customer_name}, and more to personalize the WhatsApp message for each order.

= What is MyD Notifications? =
MyD Notifications is a paid add-on that sends automatic WhatsApp messages to customers at every order status change. It works natively with this plugin. [See plans and pricing →](https://myddelivery.com/notifications/)

Visit [eduardovillao.me](https://eduardovillao.me/wordpress-plugins/) for documentation and support.
